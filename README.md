## Features

- **Draw Characters**: Use an intuitive canvas interface to draw each letter, number, and symbol
- **Complete Character Set**: Support for uppercase, lowercase, numbers, and common symbols
- **Drawing Tools**: Pen and eraser tools with adjustable brush sizes
- **Live Preview**: See your custom font in action as you create it
- **Export to TTF**: Download your font as a .ttf file to use in any application
- **No Backend Required**: Everything runs in your browser using JavaScript and opentype.js

## How to Use

1. **Open the App**: Simply open `index.html` in a web browser
2. **Select a Character**: Click on any character button in the sidebar (starts with 'A')
3. **Draw**: Use your mouse or touchscreen to draw the character on the canvas
4. **Adjust Tools**:
   - Change brush size with the slider
   - Switch between pen and eraser
   - Clear the canvas if needed
5. **Save**: Click "Save Character" to store your drawing
6. **Repeat**: Click "Next" or select another character to continue
7. **Preview**: Type in the preview box to see your font in action
8. **Export**: Once you've drawn enough characters, enter a font name and click "Download Font"

## Installation

No installation required! Just clone or download this repository and open `index.html` in any modern web browser.

```bash
git clone https://github.com/yourusername/font-maker.git
cd font-maker
# Open index.html in your browser
```

## Technology Stack

- **HTML5 Canvas**: For drawing interface
- **Vanilla JavaScript**: Core application logic
- **opentype.js**: Font generation and export
- **CSS3**: Modern, responsive styling

## Tips for Best Results

- Draw characters centered on the canvas
- Use the baseline guide (horizontal line) for consistent letter alignment
- Keep your strokes smooth and clear
- Start with uppercase A-Z for a basic working font
- You can always go back and edit any character
